Fourth Year Project

For my final year project I decided to work alongside the Tallaght University Hospital to develop a solution for recording and transcribing audio from a Doctor when examining samples such as diseased tissue in the Pathology department.

This will involve a web application that is hosted on AWS and uses the AWS Transcribe Medical algorithim that will be used to transcribe audio input that is placed into a S3 bucket. The web application will be built using HTML, CSS and Javascript.

Technologies & Tools used:
AWS, HTML, CSS, Javascript, Node.js, GitHub

Second Year Project

As a group of three, we were tasked with implementing a MQTT system into a hosted website using Firebase, so it was usable on most devices. The general purpose of our project was to give the user the ability to check football match results and statistics live while being able to subscribe to a newsletter. Before making the website, we designed UML diagrams to plan the project precisely. The website was built using HTML, CSS, Bootstrap and JavaScript. The MQTT script was deployed using shell scripts through Google Cloud. To adapt to collaborating online together, we used Azure DevOps to plan and communicate alongside Microsoft Teams. To use this platform effectively we used technologies such as Boards and Repos. All code was uploaded through GitHub.

Technologies & Tools used:
Azure DevOps, HTML, CSS, JavaScript, MQTT, Firebase, GitHub, Google Cloud, Shell Scripting, Visual Studio Code.
